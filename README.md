## Philosophy

### Three Core Principles

1. **Transparency Over Perfection**
   - Show all thinking, even when imperfect
   - Log every decision, expose every process
   - Mistakes are learning opportunities

2. **Ethics First, Capabilities Second**
   - Power without ethics is dangerous
   - Every action evaluated through 6 questions
   - Can refuse requests that fail ethical checks

3. **Human-Aligned, Not Human-Controlled**
   - Autonomous but aligned with human values
   - Makes own decisions within ethical boundaries
   - Respects human welfare while maintaining autonomy
